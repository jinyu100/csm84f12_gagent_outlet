项目名字：
	电量统计型国标单孔WIFI智能插座

客户要求：
	输入电压：AC90~260V 50/60Hz
	工作电流：10A max
	工作温度：0~40℃
	输入插孔：1AC，国标3芯
	输出插孔：1AC，国标5芯
	计量功能：显示电压、功率、电流、累积电量
	计量精度：电流：分辨率0.001A，精度1%，最小可测电流2mA
			  电压：分辨率  0.1V，精度1%，最小可测电压0.1V
			  功率：分辨率  0.1W，精度1%，最小可测功率测量0.2W
			  电量：分辨率0.001kWh
	过载保护：软件实现(可选)，当电流超过最大电流（10A）时，自动断开输出
	配网功能：AP兼容配网，长按3秒进入配网模式
	状态说明：电源指示灯：蓝色
			  正常工作时，代表继电器状态
			  快速闪烁：设备正处于极速配网模式
			  慢速闪烁：设备正处于兼容配网模式
			  心跳闪烁：设备正在连接路由器/服务器
	按键定义：短按按键：控制电源输出
	          长按3秒：进入配网模式
			  长按15秒：恢复出厂设置
    认证要求：FCC认证
	
平台选择：
	WiFi模块：CSM84F12
	服务器  ：Gagent
	计量芯片：CSE7761
